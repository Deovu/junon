Junon.io
=======

Installation
--------
1. Install Node.js 16.15.0 (ideally using [nvm](https://github.com/nvm-sh/nvm) )
2. Install project dependencies
    cd ~/junon
    npm install

Running
--------
1. Run client
    npm run client

2. Run server
    npm run dev

Debugging
---------

    npm run inspect

    Enter "chrome://inspect/" in chrome browser
    Click "Open dedicated DevTools for Node"
